MedBiquitous Activity Report Schema


Description

The MedBiquitous Activity Report Schema provides a data structure that allows one to represent professional education and certification related accomplishments of an individual in a standard format. This structure then enables the tracking of education and certification activities, including point of care learning, across organizational boundaries.

Schema

Version 1 of the schema file activityreport.xsd is available at: http://ns.medbiq.org/activityreport/v1/activityreport.xsd

Specification

The schema specification is available at: http://www.medbiq.org/working_groups/activity_report/ActivityReportSpecification.pdf

For more information, visit the MedBiquitous website.

MedBiquitous Educational Trajectory Extensions Schema


Description

The MedBiquitous Educational Trajectory Extensions Schema provides definitions for extensions to the JISC LEAP2A specification necessary for the MedBiquitous Educational Trajectory Profile.

Schema

The schema file educationaltrajectoryextensions.xsd is available at: http://ns.medbiq.org/educationaltrajectory/v1/educationaltrajectoryextensions.xsd

Specifications

A draft version of the Educational Trajectory specification is available at: http://www.medbiq.org/sites/default/files/EducationalTrajectoryProfile.pdf

Note: The Educational Trajectory is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

MedBiquitous Virtual Patient Data Schema


Description

The MedBiquitous Virtual Patient Data Schema provides a data structure that allows one to represent the clinical and demographic data associated with a virtual patient in a standard format. This structure then enables institutions to share virtual patient data alone or as part of a more sophisticated, media rich virtual patient.

Schema

The draft schema file virtualpatientdata.xsd is available at: http://ns.medbiq.org/virtualpatientdata/v1/virtualpatientdata.xsd

A zip file of all MedBiquitous Virtual Patient schemas is available at: http://ns.medbiq.org/virtualpatientdata/v1/virtualpatient_schemas.zip

Specifications

The Virtual Patient Data specification, which describes virtual patient data, data availability model, and activity model schema files, is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientDataSpecification.pdf

The Virtual Patient Player specification is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientPlayerSpecification.pdf

Note: The Virtual Patient Data Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

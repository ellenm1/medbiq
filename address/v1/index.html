MedBiquitous Address Schema


Description

The MedBiquitous Address Schema provides a data structure that allows one to represent an address in a standard format. It can be used alone or imported into other XML schema that must represent addresses. It is designed to be highly reusable.

Schema

Version 1 of the schema file address.xsd is available at: http://ns.medbiq.org/address/v1/address.xsd

Specification

The schema specification is available at: http://www.medbiq.org/working_groups/professional_profile/AddressSpecification.pdf

For more information, visit the MedBiquitous website.

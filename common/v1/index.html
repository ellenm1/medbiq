MedBiquitous Common Schemas


Description

Schemas in the MedBiquitous Common namespace may be used as common datatypes and elements by MedBiquitous standards and specifications.Schemas in the Common namespace include:

  • attachment.xsd, available at: http://ns.medbiq.org/common/v1/attachment.xsd
    For more information, see the MedBiquitous Guidelines for Transmitting Binary Attachments.

For more information, visit the MedBiquitous website.

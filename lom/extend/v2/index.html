MedBiquitous Healthcare Learning Object Metadata Schemas

Description

The MedBiquitous Healthcare Learning Object Metadata (Healthcare LOM) Schemas provide a data structure that allows one to describe learning content in a standard format to facilitate search and discovery. Healthcare LOM customizes the IEEE Learning Object Metadata standard included in the SCORM reference model.

Schema

Iterative drafts for version 2.0 of the schema files are contained within a zip file available at: http://ns.medbiq.org/lom/extend/v2/healthcarelom.zip

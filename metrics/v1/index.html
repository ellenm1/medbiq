MedBiquitous Medical Education Metrics Schema

Description

The MedBiquitous Medical Education Metrics Schema provides a data structure that allows one to represent aggregate evaluation data for a particular health professions learning activity in a standard format. It enables educators to exchange evaluation data with grantors, supporters, accreditors, and other organizations.

Schema

Version 1 of the schema file medicaleducationmetrics.xsd is available at: http://ns.medbiq.org/metrics/v1/medicaleducationmetrics.xsd

Specification

The schema specification is available at: http://www.medbiq.org/working_groups/metrics/MedicalEducationMetricsSpecifications.pdf

For more information, visit the MedBiquitous website.

MedBiquitous Performance Framework Schema


Description

The MedBiquitous Performance Framework Schema provides a consistent format and data structure for defining a performance framework, the levels of performance used to assess competency. This, combined with other existing and emerging specifications, enables learner's competency data to be portable across systems.

Schema

A draft version of the schema file performanceframework.xsd is available at: http://ns.medbiq.org/performanceframework/v1/performanceframework.xsd

Specifications

The Performance Framework specification will be available soon.

Note: The Performance Framework Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

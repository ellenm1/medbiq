MedBiquitous Data Availability Model Schema


Description

The MedBiquitous Data Availability Model Schema provides a data structure that allows one to represent which virtual patient data elements and media resources are available at a particular node of a virtual patient activity. This structure is one component of the MedBiquitous Virtual Patient Architecture.

Schema

A draft version of the schema file dataavailabilitymodel.xsd is available at: http://ns.medbiq.org/dataavailabilitymodel/v1/dataavailabilitymodel.xsd

A zip file of all MedBiquitous Virtual Patient schemas is available at: http://ns.medbiq.org/virtualpatientdata/v1/virtualpatient_schemas.zip

Specifications

The Virtual Patient Data specification, which describes virtual patient data, data availability model, and activity model schema files, is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientDataSpecification.pdf

The Virtual Patient Player specification is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientPlayerSpecification.pdf

Note: The Data Availability Model Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

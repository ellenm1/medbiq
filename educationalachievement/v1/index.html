MedBiquitous Educational Achievement Schema


Description

The MedBiquitous Educational Achievement Schema provides a common format for data documenting competence and other educational achievements.

Schema

The schema file educationalachievement.xsd is available at: http://ns.medbiq.org/educationalachievement/v1/educationalachievement.xsd.

Specifications

A draft version of the Educational Achievement specification will be available soon.

Note: The Educational Achievement specification is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

MedBiquitous Survey Items Schema

Description

The MedBiquitous Survey Items Schema provides a data structure that allows one to represent multiple learner survey items in a standard format. It enables educators to develop repositories of survey items and to link evaluation data back to these uniquely identified survey items. That in turn enables the compilation of evaluation data across organizations and systems for research and improvement.

Schema

The schema file surveyitems.xsd is available at: http://ns.medbiq.org/surveyitems/v1/surveyitems.xsd

Specification

The schema specification is available at: http://www.medbiq.org/working_groups/metrics/SurveyItemsSpecifications.pdf

Note: The MedBiquitous Survey Items Schema and Specification are not yet final and are subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

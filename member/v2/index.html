MedBiquitous Healthcare Professional Profile Schema


Description

The MedBiquitous Healthcare Professional Profile Schema provides a data structure that allows one to represent one or more healthcare professionals in a standard format. This structure then enables the exchange of this data between related business systems and over the Web.

Schema

Adraft version of version 2 of the schema file member.xsd is available at: http://ns.medbiq.org/member/v2/member.xsd

Specification

The schema specification is available at: http://www.medbiq.org/working_groups/professional_profile/ProfessionalProfileSpecifications_pointrelease.pdf

For more information, visit the MedBiquitous website.

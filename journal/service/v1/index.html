MedBiquitous Journal Services


Description

The MedBiquitous Journal Services provide a common Web services interface needed to connect journal aggregators with one another and the community they serve. They allow organizations to exchange lists of journals, journal issues, and articles within an issue. Journal Services are based on Web Services Description Language (WSDL).

Web Services Descriptions

Web services descriptions are divided into three files: journaltypes.xsd, journalbindings.wsdl, and journalservice.wsdl. All files are available in this directory. The version 0.2 Journal Services files may be downloaded from the following zip file: http://ns.medbiq.org/journal/service/v1/journalservices.zip

Specification

The Journal Services specification is available at: http://www.medbiq.org/working_groups/journal/JournalServicesSpecification.pdf

For more information, visit the MedBiquitous website.

MedBiquitous Name Schema


Description

The MedBiquitous Name Schema provides a data structure that allows one to represent a person name in a standard format. It can be used alone or imported into other XML schema that must represent person names. It is designed to be highly reusable.

Schema

The point release of the schema file name.xsd is available at: http://ns.medbiq.org/name/v2/name.xsd

For more information, visit the MedBiquitous website.

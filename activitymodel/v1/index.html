MedBiquitous Activity Model Schema


Description

The MedBiquitous Activity Model Schema provides a data structure that allows one to represent the learning activity associated with a virtual patient in a standard format. In the MedBiquitous Virtual Patient Architecture, Activity Model is the component that brings Virtual Patient Data, Data Availability Model, and Media Resources together in to a cohesive learning experience. This structure then enables institutions to share virtual patient data alone or as part of a more sophisticared, media rich virtual patient.

Schema

A draft schema file activitymodel.xsd is available at: http://ns.medbiq.org/activitymodel/v1/activitymodel.xsd

Specifications

The Virtual Patient Data specification, which describes virtual patient data, data availability model, and activity model schema files, is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientDataSpecification.pdf

The Virtual Patient Player specification is available at: http://www.medbiq.org/working_groups/virtual_patient/VirtualPatientPlayerSpecification.pdf

For more information, visit the MedBiquitous website.

MedBiquitous Competency Object Schema


Description

The MedBiquitous Competency Object Schema provides a consistent format and data structure for defining a competency object. This, combined with other existing and emerging specifications, enables educational resources and activities to be tied to a competency framework.

Schema

A draft version of the schema file competencyobject.xsd is available at: http://ns.medbiq.org/competencyobject/v1/competencyobject.xsd

Specifications

A draft version of the Competency Object specification is available at: http://www.medbiq.org/sites/default/files/CompetencyObjectSpecification.pdf

Note: The Competency Object Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

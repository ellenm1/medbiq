MedBiquitous Competency Framework Schema


Description

The MedBiquitous Competency Framework Schema provides a consistent format and data structure for defining a competency framework. This, combined with other existing and emerging specifications, enables educational resources and activities to be tied to a competency framework.

Schema

A draft version of the schema file competencyframework.xsd is available at: http://ns.medbiq.org/competencyframework/v1/competencyframework.xsd

Specifications

The Competency Framework specification is available at: http://www.medbiq.org/working_groups/competencies/index.html

Note: The Competency Framework Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.

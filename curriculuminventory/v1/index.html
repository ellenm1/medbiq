MedBiquitous Curriculum Inventory Schema


Description

The MedBiquitous Curriculum Inventory Schema provides a consistent format and data structure for the exchange of curriculum data for benchmarking and educational research.

Schema

A draft version of the schema file curriculuminventory.xsd is available at: http://ns.medbiq.org/curricluminventory/v1/curriculuminventory.xsd
A sample XML file is also available.

Specifications

The Curriculum Report specification is available at: http://www.medbiq.org/sites/default/files/files/CurriculumInventorySpecification.pdf

Note: The Curriculum Report Schema is not yet final and is subject to change as a result of the standards development process.

For more information, visit the MedBiquitous website.
